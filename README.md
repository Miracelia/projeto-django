🛒 Sistema de Cadastro — Produtos e Vendedores

Sistema web desenvolvido em Django para cadastro e gerenciamento de produtos e vendedores.


📋 Funcionalidades

👤 Cadastro de Vendedores


Cadastrar, editar e excluir vendedores
Campos: Nome, Telefone, E-mail e CPF
Validação de CPF (algoritmo oficial)
Formatação automática de CPF e telefone
Busca por nome ou CPF
Interface dark mode moderna


📦 Cadastro de Produtos


Cadastrar produtos com nome, data e categoria
Categorias: Termogênicos, Proteicos e Vitaminas
Interface integrada ao mesmo padrão visual
